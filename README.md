# Wine-Explorer
A web application with voice input support that answers questions about a wine dataset. <br>
<br>

## How to Use
Put the file in the same directory and run the wine-voice-app.html file in the browser.
<br>
The files need to be served together which can be done by running <br>
> python3 -m http.server 8080 <br>

while in the directory, and then open http://localhost:8080 in browser. 
<br>
After running the wine-voice-app.html file, put an Anthropic API key, and then type the question or speak by pressing the microphone icon.
